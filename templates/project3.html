In the absence of forcing A = 0, the downward pendulum y = 0 is a stable equilibrium, and the inverted pendulum y = pi is an unstable equilibrium.

Two examples are shown below.

Two figures above show the moving pendulum when y = 0 and y= pi.

Videos show two balls at the same position for a whole time.

Downward pendulum at the left is a stable equilibrium.

Inverted pendulum at the right is an unstable equilibrium.

In this problem, y is set as 3.1, and find the range of parameter A for which the inverted pendulum becomes stable.

Unstable inverted pendulum

Stable inverted pendulum

To find the range of parameter A, bisection method was used to find the value as accurate as possible.

Note that when the inverted pendulum is stable, then the ball never goes downward.

The function of a parameter A is defined in a matlab code that gives +1 or -1.

If the ball goes downward, then the function gives -1.

If the ball never goes downward throughout the whole time period, the function gives +1.

In conclusion, the range of parameter A was calculated as 18.333816137413876 <= A <= 25.677509645001138
